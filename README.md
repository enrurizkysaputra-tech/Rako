# RAKO — Reservasi Antrian Klinik Online

Aplikasi reservasi klinik online berbasis Laravel 12 dengan fitur:
- Autentikasi & manajemen role (Admin, Dokter, Pasien)
- Manajemen data dokter & jadwal praktik (CRUD)
- Reservasi klinik dengan validasi kuota slot
- Nomor antrian & estimasi jam kedatangan otomatis
- Rekam medis digital
- Dashboard statistik untuk Admin

## Tech Stack
- Laravel 12 (PHP 8.2+)
- Laravel Breeze (Autentikasi)
- Bootstrap 5
- SQLite / MySQL
